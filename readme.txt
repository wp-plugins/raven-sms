Send a SMS via WordPress using Airtelco srl services.

== Description ==
Wordpress implementation for Airtelco API

= Features =

* Send SMS to number(s) and addressbook.
* Show credit.
* Show delivery notification.
* Widget for register user to sms addressbook.
* Import/Export contacts.

= Translators =
* Italian

= Support =
* support@airtelco.com

== Installation ==
1. Upload `wp-ravensms` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
